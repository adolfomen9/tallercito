
# 🏦 Taller de Gestión del Conocimiento (KM) - Banco Tecnológico

**Curso:** Administración de Tecnologías de Información  
**Ciclo:** IX Ciclo, Ingeniería en Sistemas  
**Caso Práctico:** Reducción de la dependencia del Administrador de Red Senior.

---

## 📋 Diagnóstico del Problema
El "Banco Tecnológico" enfrenta un alto riesgo operativo debido a la **dependencia de una sola persona** (Administrador de Red Senior). Si esta persona no está disponible, el equipo de TI no puede resolver fallos críticos en la red, afectando la continuidad del negocio.

## 💡 Solución de Gestión del Conocimiento (KM) implemented
Se ha creado este repositorio para actuar como la **Wiki de TI**, centralizando:
1. Procedimientos detallados de configuración de la red.
2. Scripts de verificación y automatización.
3. Troubleshooting y resolución de incidentes paso a paso para personal Junior.
